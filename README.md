## Install

environment require

```bash
node <= 10
npm i
```

## Run

```bash
npm start
```

## Modify

change any `JavaScript` codes in the path `app/js/es6` and `stylesheet` in the file `app/css/main.scss`, it will be autocompiled by gulp.
